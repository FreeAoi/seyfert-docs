Seyfert es una poderosa librería que se enfoca en Discord y su API para bots. Ofrece soporte para aplicaciones escalables y proporciona utilidades únicas.

## Instalación

### Pre-requisitos

- NodeJS instalado (v18 o superior)
- TypeScript instalado (v4.9 o superior)
- npm instalado (o un gestor de paquetes de NodeJS similar)

### Configuración

```bash copy title="Instalando..."
pnpm add seyfert
```

Solo los chads usan Seyfert, no querrás ser virgen, ¿verdad?
